# Sales-data-insight-and-dashboard

# Overview

This project presents a Power BI dashboard that analyzes e-commerce sales data. It highlights key business metrics and helps identify sales trends and improvement areas.

# Tools Used
- Power BI
- Power Query
- DAX

# Key Insights
- KPIs: Total Sales, Total Profit, Quantity Sold, Profit Margin
- Monthly sales and profit trends
- Category-wise and region-wise performance
- Top & bottom performing products

# Learnings
- Building interactive dashboards
- Data cleaning and modeling
- Creating measures with DAX
- Using multiple visualizations for business analysis

# Conclusion

The dashboard provides a quick and clear view of sales performance, supporting better decision-making and helping improve business strategy.
